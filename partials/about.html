About the Open Siddur client demo

The Open Siddur Project intends to create a complete end to end system for producing high quality Jewish prayer books, for paper publication or sharing online. We are committed to our Jewish tradition, free culture and open source.

Source code for the client and server are available from our github repositories. Unless otherwise indicated, code is licensed under the GNU Lesser General Public License, version 3 or later. Content may be licensed under any of the Creative Commons free culture licenses.

To report issues with this demo or to get involved in the technical aspects of the project, contact the opensiddur-tech Google group.

About this demo

This is demo version 0.0.6. It supports:

  • user registration and editing of first and third party user profiles
  • a bibliography editor
  • a basic XML editor for JLPTEI original texts
  • very basic sharing permissions: public or restricted
  • aligning two parallel texts, such as translations, to link them together
  • basic editing of conditionals documents
  • basic editing of annotations documents (new in this version)
